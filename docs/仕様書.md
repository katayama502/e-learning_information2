# リスキル大学 Eラーニングシステム 仕様書

**バージョン**: 1.0.0  
**最終更新**: 2026-04-25  
**プロジェクト**: e-learning (EIS リスキル大学)

---

## 1. システム概要

### 1.1 目的

「リスキル大学」は、DX・AI・ChatGPT・Gemini・Claude 等の最新テクノロジーに関するリスキリング教材を提供する Eラーニングプラットフォームです。ユーザーが YouTube 動画を中心とした講義コンテンツを視聴し、学習進捗を管理できます。

### 1.2 対象ユーザー

| ユーザー種別 | 説明 |
|------------|------|
| 一般学習者 (seeker) | コースを受講する一般ユーザー |
| 管理者 (admin) | コンテンツ管理・ユーザー管理を行う管理者 |

### 1.3 主要機能

- **学習トラック閲覧**: DX・AI・ChatGPT 等のカテゴリ別学習トラック
- **コース一覧・詳細**: モジュール構成でコースを閲覧
- **レッスン視聴**: YouTube 動画プレイヤーによる講義視聴（全画面対応）
- **進捗管理**: 受講済みレッスンの記録・全体進捗表示
- **管理者機能**: コース・モジュール・レッスンの CRUD、CSV インポート、AI クイズ生成

---

## 2. 機能仕様

### 2.1 学習者向け機能

#### 2.1.1 ダッシュボード (`/reskill`)

- 学習トラック一覧を表示
- 各トラックの進捗率をプログレスバーで表示
- 全体進捗（完了レッスン数 / 全レッスン数）を表示
- 続きから視聴できるクイックアクセスリンク

#### 2.1.2 コース一覧 (`/reskill/courses`)

- 全公開コース（`course_curriculums`）を一覧表示
- カテゴリフィルタリング機能
- コース別の説明・レッスン数・概要表示

#### 2.1.3 コース詳細 (`/reskill/courses/[id]`)

- コースのモジュール構成を表示
- 各レッスンへのリンク
- 受講済みレッスンのチェックマーク表示

#### 2.1.4 レッスン視聴 (`/reskill/lesson/[id]`)

- YouTube 埋め込みプレイヤーによる動画再生
- 左サイドバーにレッスン一覧（コース内ナビゲーション）
- レッスン完了マーク機能（Zustand + localStorage 永続化）
- 前後レッスンへのナビゲーション

### 2.2 管理者向け機能

#### 2.2.1 トラック管理 (`/admin/elearning`)

- 学習トラック（`courses` テーブル、`category='Track'`）の作成・編集・削除
- トラックへのコース（モジュール）割り当て
- 公開・非公開切り替え

#### 2.2.2 コンテンツ管理 (`/admin/elearning/content`)

- モジュール（`course_curriculums`）の作成・編集・削除
- レッスン（`course_lessons`）の追加・編集（YouTube URL 指定）
- CSV 一括インポート機能
- AI クイズ自動生成機能（Anthropic Claude API 使用）

### 4.2 リレーション

```
courses (1) ──< course_curriculums (1) ──< course_lessons
                     (モジュール)              (レッスン)
   (トラック)
```

### 4.3 Row Level Security (RLS)

| テーブル | SELECT | INSERT/UPDATE/DELETE |
|---------|--------|----------------------|
| `courses` | 全ユーザー（公開のみ） | admin のみ |
| `course_curriculums` | 全ユーザー（is_public=true） | admin のみ |
| `course_lessons` | 全ユーザー | admin のみ |
| `profiles` | 本人のみ | 本人のみ |

## 5. 認証仕様

### 5.1 認証方式

- **プロバイダー**: Supabase Auth（メール + パスワード）
- **セッション管理**: `@supabase/ssr` による HttpOnly Cookie 方式
- **クライアント状態**: Zustand (`useAppStore`) に `authStatus` / `activeRole` を保持

### 5.2 認証フロー

1. `/login` ページでメール・パスワード入力
2. Supabase `signInWithPassword()` で認証
3. Server Action `loginAction()` で HttpOnly Cookie を設定
4. `profiles.user_type` を確認し Zustand に role をセット
5. 認証済みの場合 `/reskill` へリダイレクト

### 5.3 ロール制御

| ロール | `profiles.user_type` | アクセス可能ページ |
|-------|---------------------|-----------------|
| `seeker` | 'seeker' | /reskill/**, /reskill/courses/** |
| `admin` | 'admin' | /reskill/**, /admin/elearning/** |

管理者ページのアクセス制御は `middleware.ts`（Supabase SSR ミドルウェア）で行い、`user_type != 'admin'` の場合は `/login` にリダイレクトします。

## 6. API 仕様

### 6.1 Eラーニング API

| メソッド | エンドポイント | 説明 |
|---------|-------------|------|
| GET | `/api/elearning/tracks` | 学習トラック一覧取得 |
| GET | `/api/elearning/tracks/[id]` | トラック詳細取得 |
| GET | `/api/elearning/tracks/[id]/courses` | トラック内コース一覧取得 |
| GET | `/api/elearning/modules` | モジュール一覧取得（公開のみ） |
| GET | `/api/elearning/modules/[id]` | モジュール詳細取得 |
| GET | `/api/elearning/content` | レッスン一覧取得 |
| GET | `/api/elearning/content/[id]` | レッスン詳細取得 |
| POST | `/api/elearning` | コース作成（管理者） |
| PUT | `/api/elearning` | コース更新（管理者） |
| DELETE | `/api/elearning?id=[id]` | コース削除（管理者） |

### 6.2 クイズ生成 API

| メソッド | エンドポイント | 説明 |
|---------|-------------|------|
| POST | `/api/quiz/generate` | Anthropic Claude API でクイズ自動生成 |

### 6.3 API レスポンス形式

```json
// 成功
{ "id": "...", "title": "...", ... }

// エラー
{ "error": "エラーメッセージ" }
```

## 9. 非機能要件

### 9.1 パフォーマンス

- 初期表示: 3秒以内
- API レスポンス: 1秒以内
- YouTube プレイヤーは iframe 埋め込みのため、Google のネットワーク依存

### 9.2 対応環境

| カテゴリ | 対応範囲 |
|---------|---------|
| ブラウザ | Chrome, Safari, Firefox, Edge（最新2バージョン） |
| デバイス | PC (1024px〜), タブレット (768px〜), スマートフォン (〜767px) |
| OS | Windows, macOS, iOS, Android |

### 9.3 セキュリティ

- 全通信: HTTPS 強制
- 認証: Supabase Auth（JWT + HttpOnly Cookie）
- DB アクセス: RLS による行レベルセキュリティ
- 管理者ページ: ミドルウェアレベルで保護
